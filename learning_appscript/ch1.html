Глава 1. Введение в Google App Script

Библиотека clasp

clasp— это инструмент командной строки, который позволяет разработчикам работать с проектами Google Apps Script (GAS) локально, используя редакторы кода и системы контроля версий, такие как Git. Он упрощает процесс разработки, позволяя создавать, развертывать и управлять проектами Apps Script из командной строки.

Чтобы работать с clasp надо включить API у проекта. Это можно сделать по ссылке https://script.google.com/home/usersettings.

npm install -g @google/clasp //установить библиотеку clasp глобально
clasp login //залогиниться в GAS
clasp open // открыть проект GAS
clasp clone <id> //клонировать проект в локальную область
clasp pull //выпулить проект в локальную область
clasp push //запушить локальную версию проект на GAS

О других функциях пакета можно узнать в README-разделе GitHub'а библиотеки.
