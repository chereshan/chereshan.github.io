Как показать/скрыть элемент постепенно

Мы можем передать в методы hide(), show() и toggle() аргументы - время в милисекундах и функция, которая будет вызвана после заверщения процесса. 

<script>
        $('.ex2.hide').click(function(){
            $('#divex2').hide($('input.ex2').val(), function(){
                $('#divex2 p:first')
            })
        })
        $('.ex2.show').click(function(){
            $('#divex2').show($('input.ex2').val(), function(){
                $('#divex2 p:first')
            })
        })
        $('.ex2.toggle').click(function(){
            $('#divex2').toggle($('input.ex2').val(), function(){
                $('#divex2 p:first')
            })
        })
</script>

Изменение прозрачности элемента

Мы можем управлять прозразностью некоторого элемента кликом на него с помощью метода fadeTo():

  <script>
      $('p.ex5:first').click(function() {
          $( this ).fadeTo( "slow", 0.33 );
      });
  </script>

Hover (Эффекты наведения мыши)

Ниже приведен квадрат, цвет которого случайно меняется при наведении курсора:

  <script>
      $('div.ex13').hover(function(){
              function getRandomColor() {
                  var letters = '0123456789ABCDEF';
                  var color = '#';
                  for (var i = 0; i < 6; i++) {
                      color += letters[Math.floor(Math.random() * 16)];
                  }
                  return color;
              }

              $('div.ex13').css('background',getRandomColor());
          },
          function (){
              $('div.ex13').css('background','white');
          })
  </script>

Позиционирование элементов

Нарисуем рамку с четырьмя квадратами внутри. При щелчке мышкой на один из них на вывод ниже будет выведены значения offset() и position().

Метод offset() возвращает значения отсупы top и left относительно документа.

Метод position() возвращает значения top и left относительно родительского элемента.

  <script>
      $('div.ex15.inner').click(function(){
          var o = $(this).offset();
          $("p.ex15:first span").text("left: " + o.left + "px, top: " + o.top + "px");
          var p = $(this).position();
          $(this).effect('highlight')
          $("p.ex15:last span").text("left: " + p.left + "px, top: " + p.top + "px");
      })
  </script>


Offset — left: ?px, top: ?px

Position — left: ?px, top: ?px
