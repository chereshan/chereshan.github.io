Глава 1. Введение в Android

Компоненты Android

Приложение для Android состоит из одного или нескольких компонентов, написанныъ в виде классов Java:

  1. Активность (Activity) - визуальные компоненты (представления) отдельного экрана, а также код, который выводит на экран данные и может реагировать на события, вызываемые пользователем. Почти каждое приложение имеет по крайней мере 1 класс Activity.
  2. Служба (Service)
  3. Широковещательные приемники
  4. Провайдеры контента
  5. Адаптеры синхронизации
